```markdown
# API / Функції логічного ядра

## SimulationEngine

Головний компонент системи, який відповідає за обчислення положення планет.

### Методи:

- `start()` - запуск симуляції
- `pause()` - пауза симуляції
- `updatePositions()` - оновлення координат планет
- `setStrategy(strategy)` - зміна стратегії руху планет
- `getState()` - отримати поточний стан симуляції
- `restoreState(state)` - відновити стан симуляції

### Unit-тести

- Тестування `start()` та `pause()` - перевірка стану `isRunning`
- Тестування `updatePositions()` - перевірка зміни координат планет
- Тестування `setStrategy()` - перевірка коректного встановлення стратегії
```
