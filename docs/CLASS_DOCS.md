# Документація класів

## SimulationEngine

### Властивості:

- `planets` - масив об'єктів планет
- `isRunning` - стан симуляції
- `strategy` - поточна стратегія обчислення орбіт

### Методи:

- `start()` - запускає симуляцію
- `pause()` - зупиняє симуляцію
- `updatePositions()` - оновлює координати планет за поточною стратегією
- `setStrategy(strategy)` - змінює стратегію руху
- `getState()` - повертає поточний стан
- `restoreState(state)` - відновлює стан симуляції збережений раніше
