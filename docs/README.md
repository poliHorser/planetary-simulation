# Planetary Simulation - Технічна документація

## Опис проєкту

Додаток моделює рух планет у планетарній системі. Користувач може змінювати масу планет, швидкість руху, стратегію обчислення орбіт, запускати та зупиняти симуляцію.

## Вимоги

- Node.js 18+ / TypeScript 5+
- Браузер для перегляду інтерфейсу
- npm / yarn

## Інструкція з запуску

1. Встановити залежності:

```bash
npm install
```
